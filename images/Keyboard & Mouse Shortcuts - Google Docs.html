To enable screen reader support, press shortcut ⌘+Option+Z. To learn about keyboard shortcuts, press shortcut ⌘slash.
Generally Speaking... 
 
●Command kicks off most keyboard shortcuts 
●Shift usually does the opposite (for example, if Command + Z is undo, Command + Shift + Z is 
redo) 
●Option and control are usually modifiers 
●It’s totally OK to keep mashing keys until what you want happens (you > machine) 
●Keyboard shortcuts are almost always also actions available and documented in the applications 
main menu (for example, open the File menu) 
Text Shortcuts 
 
These shortcuts work in text programs like Sublime or your favorite email client. 
 
Command + Right Arrow 
Move cursor to the ​end of a line 
Command + Left Arrow 
Move cursor to the ​beginning of a line 
Command + Shift + Right Arrow 
Select​ from current cursor location to end of the line 
Command + Shift + Left Arrow 
Select​ from current cursor location to beginning of the line 
Option + Right Arrow 
Move cursor to ​end of next word 
Option + Left Arrow 
Move cursor to ​beginning of previous word 
Option + Shift + Right Arrow 
Select from current cursor location to end of next word 
Option + Shift + Left Arrow 
Select from current cursor location to beginning of previous word 
 
Mac Shortcuts 
 
Command + Space 
Open Spotlight; start typing the name of an application or 
filename, press return to open it 
Command + Shift + Tab 
Switch to the ​previous program 
Command + ~ 
Switch to the ​next window ​of a program 
Command + Shift + ~ 
Switch to the ​previous window ​of a program 
Command + Shift + ] 
Switch to next tab (Safari, Chrome, Sublime, etc.) 
Command + Shift + [ 
Switch to the previous tab (Safari, Chrome, Sublime, etc.) 
Command + W 
Close the current window or tab 
 
Sublime Shortcuts 
 
For these shortcuts to work, you must tell Sublime that you’re working on an HTML file. So either save the 
file you’re working on as a .html file or click on “Plain Text” in the bottom right corner of Sublime and 
change the language to “HTML.” 
 
Option + Click & Drag 
Add multiple cursors vertically 
Command + Click 
Add another cursor wherever you click 
HTML Specific Shortcuts: 
 
Command + Option + . 
Like ending a sentence, close the last unopened HTML tag 
Tag Name + Tab 
Auto-complete an opening HTML tag (e.g., “p + tab” will give you 
<p></p> and “html + tab” will give you an HTML template) 
Tab 
Indent a section of highlighted code 
Shift + Tab 
Unindent a section of highlighted code
